# Relay Liquid Glass UI Direction

## Visual Mood

Relay should look like a modern macOS utility built for Tahoe-style Liquid Glass interfaces.

## Palette

- Deep blue glass
- Cyan highlights
- White chevron mark
- Transparent dark/light adaptive panels
- Subtle depth shadows

## Key Screens

1. Command Palette
2. Menu Bar Dropdown
3. Command Editor
4. Task Editor
5. Logs
6. Settings
7. Command Pack Import

## Icon Prompt

Design a macOS app icon for an app named Relay.

The icon should be a rounded square with layered Liquid Glass material, deep blue and cyan gradients, subtle internal reflections, and a bold white forward chevron symbol in the center. The chevron should suggest command execution, relay, forwarding, and momentum. The icon must feel native to modern macOS, premium, minimal, and recognizable at small sizes.
