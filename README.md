# Relay

**Native macOS Command Palette & Task Runner**

Relay lets you launch, search, and automate shell commands from anywhere on your Mac — instantly, without opening a terminal. Press a hotkey, type a few letters, hit Return.

> **Platform:** macOS 26 (Tahoe) or later  
> **Language:** Swift 6 / SwiftUI + AppKit  
> **Status:** Feature-complete

---

## Features

### Command Palette
- **Global hotkey** (⌥ Space by default, fully configurable) opens a floating Liquid Glass panel from anywhere on your Mac
- **Fuzzy search** across command name, category, tags, and aliases with usage-based ranking
- **Drag-and-drop** file paths directly onto the palette to insert quoted paths into the query

### Command Library
- 50+ pre-installed commands across 8 packs: AI / LM Studio, Development / Git, Docker, MLX, Tailscale / Network, Homebrew, Synology, System
- Full CRUD editor — name, description, category, SF Symbol icon, tags, aliases, shell, working directory, environment variables, timeout, and behaviour flags
- Import and export **Command Packs** as JSON for sharing or version control

### Execution Engine
- Runs commands via `zsh` or `bash` with full environment inheritance
- **Variable substitution** — `$Desktop`, `$Clipboard`, `$SelectedFinderFiles`, `$Date`, `$CurrentProject`, custom variables, and more
- Foreground (with live output panel) or background execution with optional macOS notifications
- Elevation via macOS system authentication dialog — no stored passwords, Touch ID supported

### Workflow / Task Runner
- Chain steps into multi-step workflows: Shell, Launch App, Quit App, Delay, HTTP Health Check, AppleScript, JavaScript (JXA), and Notification steps
- Per-step retry counts and continue-on-error flags
- Pre-installed workflows: *AI Environment Ready*, *Morning Setup*, *Docker Deploy*

### Integrations
- **Apple Shortcuts / Siri** — run any command, search the library, or open the palette via voice or Shortcuts automation
- **Finder Services** — right-click files in Finder to send their paths directly to the palette
- **Shell History Import** — scan `~/.zsh_history` / `~/.bash_history` and import commands in one step
- **Backup & Restore** — export the full library as JSON; auto-snapshots on every save (last 10 kept)

### Settings
- Configurable global hotkey with live conflict detection
- Custom variable editor (`$NAS`, `$CurrentProject`, any name you choose)
- Privileged Helper for curated root-level operations (DNS flush, DHCP renew, repair permissions, restart services)
- Data tab: backup / restore, snapshot history, iCloud-ready architecture

## Installing the pre-built app

Download `Relay.dmg` from [Releases](../../releases/latest), open it, and drag **Relay.app** to your Applications folder. See [INSTALL.md](INSTALL.md) for full instructions and first-run setup.

## Building from source

```bash
# 1. Clone
git clone https://github.com/siveesh/Relay.git
cd Relay

# 2. Build and test the Swift package (no Xcode needed)
cd RelayKit
swift build
swift test

# 3. Generate the Xcode project and build the app
cd ../App
xcodegen generate
xcodebuild -scheme Relay -configuration Release build
```

> `App/Relay.xcodeproj` is not committed — it is generated by XcodeGen. Run `xcodegen generate` inside `App/` after cloning or whenever you change `project.yml`.
