# Relay

Relay is a lightweight native macOS command palette and task runner for executing saved commands and workflows without opening Terminal.

This ZIP contains:
- Implementation prompt
- Product specification
- Architecture notes
- Liquid Glass UI and icon direction
- Sample command pack JSON
- Swift source scaffold files

Use `Prompts/Relay_Implementation_Prompt.md` with Codex, Claude Code, Gemini CLI, or another coding agent.
